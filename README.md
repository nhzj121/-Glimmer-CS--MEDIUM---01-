## Part 1 理解大数运算
1. 因为标准的整数数据类型都有大小限制，即只能处理一定范围内的整数数据。而字符数组可以存储任意长度的数字字符串，因此可以用来表示任意大小的整数。
2. 数位对齐-->逐位相加/相减-->处理进位（%10）/借位（%10）-->前一位+1/-1-->（如果需要）添加位数/处理高位0
3. 加法和减法，正常处理。乘法和除法，采用同号为正，异号为负。
## Part 2 初步实现大数运算
### Step 1 尝试存储大数
```c
#include <stdio.h>
#include <string.h>

int main()
{
    int i;
    char s1[40];
    scanf("%[0-9]",s1);  //存储大数
    i=strlen(s1);   //计算长度
    for(int j=0;j<i;j++)  //输出大数
    {
        printf("%c",s1[j]);
    }
}
```
